
# DLS Speech Processing.

Этот репозиторий содержит Jupiter ноутбуки с решением домашних заданий со третьей части курса **Deep Learning School** по обработке речи.

*Deep Learning School – учебная организация на базе Физтех-школы прикладной математики и информатики Московского физико-технического института.

## О курсе

Этот курс посвящён современным методам обработки звука и речи (Speech Processing) с использованием машинного обучения. В рамках курса изучены основные способы представления аудиосигналов, фундаментальные задачи обработки речи — автоматическое распознавание речи (ASR), синтез речи (TTS) и другие ключевые направления. Особое внимание уделяется работе с малоресурсными языками и построению моделей для неанглоязычных данных. В эру больших языковых моделей в курс включены темы Speech LLM и мультимодальных LLM, работающих со звуком и речью. Завершающие недели курса посвящены задачам в аудиодомене.
